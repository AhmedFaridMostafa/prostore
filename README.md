# Prostore - E-commerce Platform

Prostore is a modern, full-featured e-commerce platform built with the latest web technologies. It provides a robust, scalable, and user-friendly solution for online retail, including features for both customers and administrators.

## ‚ú® Features

- **Authentication:** Secure user sign-up and sign-in (including social login).
- **Product Management:** Admins can create, update, and delete products.
- **Shopping Cart:** Persistent shopping cart for authenticated users.
- **Checkout & Payments:** Multi-step checkout process with support for PayPal and Stripe.
- **Order History:** Users can view their past orders.
- **Admin Dashboard:** Centralized dashboard for managing users, products, and orders.
- **Responsive Design:** Fully responsive and mobile-friendly interface.
- **Search & Pagination:** Easily search for products and navigate through product lists.

## üöÄ Getting Started

Follow these instructions to set up the project locally.

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/prostore.git
cd prostore
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Set Up Environment Variables

Create a `.env` file in the root of the project and add the following variables.

```env
# Database
# Example for a local PostgreSQL instance
DATABASE_URL="postgresql://USER:PASSWORD@HOST:PORT/DATABASE?sslmode=prefer"

# NextAuth.js
# Generate a secret: openssl rand -base64 32
NEXTAUTH_SECRET=
NEXTAUTH_URL="http://localhost:3000"

# App
NEXT_PUBLIC_SERVER_URL="http://localhost:3000"
NEXT_PUBLIC_APP_NAME="Prostore"

# Payment Providers
# PayPal
PAYPAL_API_URL="https://api-m.sandbox.paypal.com"
PAYPAL_CLIENT_ID=
PAYPAL_APP_SECRET=
# Stripe
STRIPE_SECRET_KEY=
STRIPE_WEBHOOK_SECRET=
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=

# File Uploads (UploadThing)
UPLOADTHING_SECRET=
UPLOADTHING_APP_ID=

# Email (Resend)
RESEND_API_KEY=
SENDER_EMAIL="onboarding@resend.dev"
```

### 4. Run Database Migrations

Apply the database schema to your database.

```bash
npx prisma migrate dev
```

### 5. Seed the Database (Optional)

Populate the database with sample data.

```bash
npx prisma db seed
```

### 6. Run the Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.
